Help on Signing In

Following is a list of issues that you may face when Signing In, and the reasons and solutions to them.

  1. ERROR :: Could not connect to the server. Please try again in a moment

    You can get this message when the WSO2 Enterprise Service Bus (ESB) server is down or when it cannot be reached over the network.

  2. ERROR :: Login failed. Please recheck the user name and password and try again

    You can get this error even if you had spelt the user name and password correctly due to two reasons. Either the user name or password is in the wrong case (Note that user name and password is case sensitive) or the Signing In page is an inconsistent state. Check whether the caps lock is on and check whether you have spelt in the user name and the password correctly and try again.

    If the correct user name and the password is still failing then refresh the page and try again.
